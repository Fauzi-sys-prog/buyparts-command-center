# MVP Roadmap

## Phase 1: Data foundation

- Bring up local PostgreSQL
- Define product, variant, sync, and alert tables
- Implement API health and module endpoints
- Connect a Shopify development store

## Phase 2: Operations dashboard

- Render dashboard metrics from live API data
- Add SKU Intelligence and Inventory Control views
- Surface sync failures and low-stock alerts

## Phase 3: AI-assisted catalog work

- Add product enrichment job inputs and outputs
- Generate suggested titles, descriptions, and specs
- Store revisions for human approval

## Phase 4: Pricing intelligence

- Introduce pricing rules and margin safeguards
- Produce recommendation batches per SKU
- Add approval and rollback workflows

## Phase 5: Warehouse and growth

- Mirror facts into BigQuery
- Join ad performance to SKU economics
- Add merchant feed diagnostics and campaign prioritization
- Introduce RAG and LangChain flows for catalog and operator assist use cases
